[ mec - 6 april 2020 - kuesji koesnu (kuesji@koesnu.com) ]

 dependencies:
  python3

 installation:
  put file inside to any directory in path environment and give executable ( x ) flag.


 usage:
  mec "expr"
  or
  mec "expr" "expr" ...

 sample:
  mec "8-4*16/2"
  or
  mec "2*((6+4)*2)"
  or
  mec "8-4*16/2" "16-4/2**16"

 valid operations:
  + -> addition
  - -> subtraction
  * -> multiplication
  / -> division
  % -> remainder
  ** -> power
  & -> bitwise and
  | -> bitwise or
  ^ -> bitwise xor
